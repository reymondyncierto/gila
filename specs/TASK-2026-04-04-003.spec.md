## Context
Phase 1 — Core Crypto. This module uses the derived key from the Argon2id module (TASK-002)
to perform authenticated encryption/decryption of vault entries. Together they form the
complete crypto pipeline for Gila.

## Completion Criteria
- `encrypt` + `decrypt` roundtrip produces the original plaintext.
- Tampered ciphertext fails decryption with an error (authentication tag check).
- Each encryption produces a unique nonce (no nonce reuse).
- Unit tests cover roundtrip, tamper detection, and different payload sizes.
