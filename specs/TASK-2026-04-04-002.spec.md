## Context
Phase 1 — Core Crypto. This is the first cryptographic primitive needed by Gila.
The derived key will be used by the AES-GCM module (TASK-003) for encrypting vault entries.

## Completion Criteria
- `derive_key` produces a 32-byte key deterministically for the same password + salt.
- `generate_salt` produces unique 16-byte salts.
- Keys implement `Zeroize` and are wiped on drop.
- Unit tests pass.
