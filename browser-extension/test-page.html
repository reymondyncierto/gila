Gila Browser Extension Test Page

How to test the Gila extension:

1. Load the extension:
Open chrome://extensions → Enable "Developer mode" → Click "Load unpacked" → Select the browser-extension/ folder.


2. Start Gila desktop app:
Run npm run tauri dev from the project root. Create a master password if first run. Note the port from ~/.gila/bridge.port.


3. Configure the extension:
Click the Gila extension icon → Enter the port and token (from ~/.gila/bridge.port and ~/.gila/bridge.token) → Click "Connect".


4. Test form detection:
Open this page → The extension should detect the email and password fields. You'll see a small green "G" icon in the fields if you have saved credentials for this URL.


5. Test credential save:
Type an email and password below → Click "Sign In" → A banner should appear at the top: "Save to Gila?" → Click "Save" → Check the Gila desktop app for a toast notification.


6. Test auto-fill:
Refresh the page → The "G" icon should appear in the fields (since you saved a credential for this URL) → Click it → Select the credential → Fields auto-fill.


7. Test via popup:
Click the Gila extension icon → It should show the saved credential → Click it → Fields auto-fill.
